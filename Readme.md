# Best Trip Methods

## Sobre o projeto

O **Best Trip Methods** é um projeto web criado com o objetivo de **praticar desenvolvimento front-end**.  
A aplicação funciona como um quiz simples, onde o usuário responde algumas perguntas e, ao final, recebe uma sugestão do melhor método de viagem de acordo com suas escolhas.

Esse projeto faz parte do meu processo de aprendizado com **React**, buscando aplicar conceitos que são usados em projetos reais.

---

## Objetivo

O principal objetivo deste projeto é colocar em prática:

- React na prática
- Criação e organização de componentes
- Uso de estados e props
- Navegação entre páginas com React Router
- Estilização com Tailwind CSS
- Organização de um projeto front-end

---

1. O usuário acessa a página inicial
2. Inicia o quiz
3. Responde perguntas sobre suas preferências de viagem
4. Recebe uma recomendação ao final
